# Keithley 2600 series SMU utils

mainly original series (not -A or -B), since I only have access to a 2602.

## flash dumping
Dumping with `flashdump.py` works over rs232 slowly; GPIB reasonably fast, and possibly ethernet (untested) for ludicrous speed.
